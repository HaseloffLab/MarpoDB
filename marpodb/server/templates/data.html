Database access

Unfortunately we can't provide you with a direct access to the database that serves marpodb.io for security reasons. However, we are happy to share the database in a form of a dump, so you can restore and use it locally.

In order to do so, you will need a PostgreSQL server installed on your local machine. The binaries for Mac OS, Linux and Windows are provided at PostgeSQL downlaod page. For Mac OS we recommend using Postgres.app.

If you are not familiar with command line, we also recommend using a client that provides a graphical user interface. The extensive list of clients can be found here

Finally, you need to download a dump of the marpodb4 database, create a new database locally and restore it from the dump.
